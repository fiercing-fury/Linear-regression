# Linear Regression Project

This project demonstrates a simple **Linear Regression** model using Python's scikit-learn to predict housing prices based on features like area, number of rooms, etc.

## 📁 Dataset
- File: `Ecommerce.csv`
- Location: `data/` folder

## 📌 Project Overview
- Exploratory Data Analysis (EDA)
- Train-Test Split
- Model Training using Linear Regression
- Evaluation using MAE, MSE, RMSE, and R² Score
- Residuals Analysis with plots

## 🛠 Libraries Used
- pandas
- numpy
- matplotlib
- seaborn
- scikit-learn
